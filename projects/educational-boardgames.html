Educational Boardgames

Design / Technology

Objectives

1. To learn and go through each stages of the design cycle to solve a problem.

2. Engage in critical and creative thinking through Webb's DoK Levels.

Summary

Students use this project as a second run through the design thinking process. The goal is to design a board game that teaches computer science / programming concepts to a grade 4 class in our school. How they integrate the learning into the game is up to them.

The students work through each stage of the Design Thinking cycle to create a working prototype(s) that solves their initial design problem.
